Install via Homebrew on Mac

1. Install sitespeed.io

$ brew tap sitespeedio/sitespeedio
$ brew tap tobli/browsertime
$ brew install sitespeed.io

IMPORTANT: To install on OS X Yosemite, you need to install the coming 3.0 release (you need NodeJS and npm): 

npm install sitespeed.io@3.0.0-alpha-8 -g

2. Run

$ sitespeed.io -h

Install on Linux

1. Install PhantomJS (1.9)

2. Install/use Java 1.6 (or higher)

3. Make sure you have curl and xsltproc installed, if not install them.

4. Clone the project from Github 

$ git clone https://github.com/sitespeedio/sitespeed.io.git

or download and unpack the zip.

4. Run

$ bin/sitespeed.io -h

Install on Windows

First of all: all files need to be in your PATH, if you don't know how to do that read more here.

1. Install the latest full version of Git Bash

2. Install Java 1.6 or higher

3. Install PhantomJS

4. Download or clone sitespeed.io from Github

If you want to collect Navigation Timing Data you should also install:

5. Internet Explorer Driver

6. ChromeDriver

7. xsltproc for Windows. You will need the following: libxml2.dll, iconv.dll, zlib.dll, xsltproc.exe and as always put them in your PATH.
